# Python Requests API Testing

A simple Python API automation demo using the `requests` library.

## Purpose
This repository demonstrates basic API testing skills using Python + `requests`. 
It is intended to showcase clean, readable test code for QA Automation roles.

## Features
- Simple GET and POST request examples
- Status code validation
- Basic assertions
- Uses a free public test API (JSONPlaceholder)

## Tech Stack
- Python 3
- `requests` library
- pytest (optional for running tests)
